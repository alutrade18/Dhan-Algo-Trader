MODULE 9: POSTBACK WEBHOOK RECEIVER
=============================================================

File: artifacts/api-server/src/routes/postback.ts

POST /api/postback — Dhan sends order updates here

```typescript
router.post("/postback", async (req, res) => {
  const payload = req.body; // DhanOrderUpdate
  
  // Save to DB
  await db.insert(postbackLogsTable).values({ ...payload, receivedAt: new Date() });
  
  // Emit to frontend via Socket.io
  io.emit("order:update", payload);
  
  // Update trade_logs if this order was from a strategy
  if (payload.correlationId) {
    await db.update(tradeLogsTable)
      .set({ status: payload.orderStatus === "TRADED" ? "executed" : payload.orderStatus.toLowerCase(), 
             orderId: payload.orderId })
      .where(eq(tradeLogsTable.correlationId, payload.correlationId));
  }
  
  // Send Telegram alert
  if (payload.orderStatus === "TRADED") {
    await telegramService.sendAlert("ORDER_TRADED", payload);
  } else if (payload.orderStatus === "REJECTED") {
    await telegramService.sendAlert("ORDER_REJECTED", payload);
  }
  
  res.status(200).json({ status: "ok" });
});
```

Also create postback_logs table in DB for audit trail.
Show recent postback events in Settings page.
